# Viikkoraportti 3

1. Sain apua Hannulta perjantaina aiheen kanssa, joka selkeytti paljon työn toteuttamista. 
2. Ohjelma ei ole edistynyt juurikaan sillä ohjauksen jälkeen en ole ehtinyt sitä oikein työstämään.
3. Opin ohjauksessa paljon Markovin ketjun toteuttamisesta lausegeneraattorina.
4. Se miten saan toteutettua oman ohjelman huolettaa vielä, mutta kädet saveen ja pyydän tarvittaessa ohjausta.
5. Seuraavaksi alan tekemään koodia ja kun peruskoodi alkaa olemaan kasassa, teen testit siihen.
